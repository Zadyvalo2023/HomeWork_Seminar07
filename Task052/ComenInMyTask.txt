Задача 52. Задайте двумерный массив из целых чисел. Найдите среднее арифметическое элементов в каждом столбце.

Например, задан массив:
1 4 7 2
5 9 2 3
8 4 2 4
Среднее арифметическое каждого столбца: 4,6; 5,6; 3,6; 3.

=============================================================================================


Пояснение кода: 
1. Сначала определен двумерный массив  array  с помощью инициализации значений в фигурных скобках. В данном случае, это массив размером 3x4. 
2. Затем определены переменные  rows  и  columns , которые хранят количество строк и столбцов в массиве. 
3. Создан массив  columnAverages  для хранения средних арифметических значений каждого столбца. 
4. Во внешнем цикле  for  перебираются столбцы массива. Во внутреннем цикле  for  происходит суммирование значений элементов в каждом столбце. 
5. После завершения внутреннего цикла, вычисляется среднее арифметическое значение для текущего столбца и сохраняется в массиве  columnAverages . 
6. После завершения внешнего цикла, выводятся средние арифметические значения каждого столбца на консоль с помощью цикла  for  и метода `ToString("F1")`, чтобы они отображались с одной десятичной цифрой после запятой.
